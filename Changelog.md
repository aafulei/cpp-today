## [0.1.0] - 2025-05-12

### Added
- Initial release of the `today` CLI program that prints the current date in `YY/MM/DD = Www` format.
- Integrated `clang-format` configuration for code formatting.
- Editor tooling support via `clangd`.
- Simple `make` build system supporting release and debug variants.
- Basic test script `test.sh` with automated execution via GitHub Actions.
- GitHub Actions CI/CD workflows for format checks, build, and tests.
